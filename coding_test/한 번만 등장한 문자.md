## 문제:  
**문자열 s가 매개변수로 주어집니다. s에서 한 번만 등장하는 문자를 사전 순으로 정렬한 문자열을 return 하도록 solution 함수를 완성해보세요. 한 번만 등장하는 문자가 없을 경우 빈 문자열을 return 합니다.**

## 키워드:
> dict

```
def solution(s):
    answer = ''    
    b = sorted(list(s))        
    a = dict.fromkeys(b,0)    
    for i in b:
        a[i] += 1 
    for k,v in a.items():
        if v == 1:
            answer += k
    return answer
```

> 1. dict에 카운트
> 2. value가 1이 아닌 것들을 제거
> 3. 사전 순으로 정렬
> 4. join으로 string 타입으로 출력
